## Installation

- #### BetterDiscord

1. Download BetterDiscord from https://betterdiscord.app/ 

2. Install it

- #### Theme

1. Open Discord

2. Go to the Settings -> BetterDiscord -> Themes

3. Click "Open Theme Folder" button

4. Move the CSS file to the opened folder

5. Activate Theme
